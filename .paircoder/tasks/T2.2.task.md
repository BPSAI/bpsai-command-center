# Objective

Save and restore Computer Chat conversations in localStorage, keyed by operator. Add session list sidebar showing past conversations with timestamps.

# Implementation Plan

- Conversations saved to localStorage on each message (keyed by `chat:{operator}:{session_id}`)
- Session list sidebar showing all conversations for current operator
- Click session to restore, "New Chat" and "Clear" buttons
- Operator isolation — different users see only their own conversations

# Acceptance Criteria

- [ ] Conversations saved to localStorage on each message (keyed by `chat:{operator}:{session_id}`)
- [ ] Session list sidebar showing all conversations for current operator
- [ ] Click session to restore conversation
- [ ] "New Chat" button creates fresh session
- [ ] "Clear" button deletes current session from storage
- [ ] Conversations survive page refresh
- [ ] Different operators see only their own conversations
- [ ] Tests: save/restore round-trip, operator isolation, new/clear actions

# Verification

- Have a conversation, refresh page → conversation restored
- Create multiple sessions → all visible in sidebar
- Log in as different user → see empty conversation list
